# Unvibe Privacy Policy

**Last updated: July 2026**

## What Unvibe processes

Unvibe is a desktop application that helps you understand code. When you select code and activate Unvibe, the following may be sent to our AI provider:

- The exact code snippet you selected
- The file name and line range
- Your chosen explanation depth
- The programming language detected

## What Unvibe does NOT do

- Unvibe never sends your entire repository or project files
- Unvibe never reads your screen or records your keystrokes outside of intentional selections
- Unvibe never runs arbitrary code, executes commands, or modifies your files
- Unvibe does not use your code for model training

## On-device secret filtering

All code selections are scanned for secrets, API keys, tokens, and credentials **before** any data leaves your Mac. If a potential secret is detected, you are shown exactly what was found and asked for consent before anything is transmitted. High-confidence secrets are blocked entirely.

## Account and sync

When you create an account, your learning history (reviews, concepts, streaks) is synced across your devices via Supabase. This data includes:

- Review timestamps and outcomes
- Concepts encountered and understood
- Streak and progress metrics

You can review, export, or delete this data at any time from within the app.

## Third-party services

AI explanations are powered by Anthropic's Claude API. No code is sent to Anthropic without your explicit action (selecting code and requesting an explanation). Anthropic does not train on your data under their API terms.

Authentication and data sync are powered by Supabase. Supabase provides the database and auth infrastructure.

## Data deletion

You can delete your account and all associated data at any time from Settings → Account → Delete account. This removes:

- All learning events and reviews
- Account credentials and session tokens
- Synced data on our servers
