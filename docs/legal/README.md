# Legal drafts

> **⚠️ DRAFTS ONLY.** Every document in this folder is an engineering-authored draft that
> describes how the product actually behaves. They are **not legal advice** and **must be
> reviewed and finalized by a qualified attorney** before any public launch, App Store
> submission, or collection of real user data. Placeholders like `[COMPANY]`, `[JURISDICTION]`,
> and `[support@…]` must be filled in.
