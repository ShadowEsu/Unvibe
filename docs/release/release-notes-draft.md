# Unvibe private beta — draft release notes

Unvibe is a macOS learning layer for understanding AI-generated code. Use the shortcut from VS Code or Cursor to review selected code, choose one of five explanation levels, ask follow-ups, test your understanding, and keep a private learning history.

This candidate adds durable local-first learning, resumable account sync, cursor-based restore beyond 500 records, cautious concept evidence (“seen,” “familiar,” “strong,” or “needs review”), one-time device authorization, account deletion, secret filtering before remote requests, and clearer offline/auth states.

Private beta constraints: Apple-silicon macOS only; VS Code/Cursor focused workflow; Accessibility permission required for cross-app selection; no mobile, browser extension, Windows/Linux, enterprise admin, billing, local model, or security certification. AI output may be wrong and must be verified against code. See the published privacy/terms and known-limitations pages before testing.
