# Private-beta polish implemented

## App experience

1. **Selection capture** — launch a review directly from the Island or tray.
2. **Clipboard fallback** — use the same safe capture path when selection access is unavailable.
3. **Private-beta metering** — 30 selected-code prompts and 20 cloud explanations per month.
4. **Learning history** — open recent explanations from the Island or companion app.
5. **Progress feedback** — streak and explanation totals remain visible in the Island.
6. **Private, local-first handling** — the local secret scan remains visible before cloud analysis.
7. **Quick access** — tray commands show, hide, open, or quit Unvibe without relying on the floating surface.
8. **Quiet behavior** — notification and quiet-hour preferences remain respected.
9. **Keyboard access** — global shortcut, Enter-to-expand, and Escape-to-collapse.
10. **Private-beta delivery** — custom purple DMG, ZIP, and Windows package scripts with checksums.

## Island experience

1. **Unclipped expanded layout** — a 194px transparent window safely contains the drawer.
2. **Dormant pulse** — one restrained purple breathing pixel when the Island is idle.
3. **Spatial morph** — corners, shadow, and border transition before drawer content appears.
4. **Blur-to-sharp entry** — drawer content settles in as one controlled unit.
5. **Action confirmation** — selected actions temporarily become clear checkmarked progress messages.
6. **Right-click menu** — contextual actions work directly on the Island.
7. **Tray parity** — the same core actions are available even when the Island is hidden.
8. **Hover preview** — last learning, streak, and totals appear without opening the full companion.
9. **Accessibility motion mode** — the system Reduce Motion preference removes animation.
10. **Keyboard interaction** — Enter opens the preview; Escape returns it to compact mode.
